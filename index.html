Squad

Ruby on Rails / JavaScript / Twilio API / AWS S3 / Travis CI / React Native

Squad was our capstone project for Dev Bootcamp, built in 8 days by a team of 4. Our idea was to build a mobile friendly application that allows a user to create a friends list and send out polls when they're out and about, needing quick advice. The user is then able to watch the feedback come back in real time via a live chart update. The main goal was for ease of use for users wanting immediate feedback. Being able to send in a response by a simple text without having the app itself open makes it more convenient and less of a hassle to reply to someone else's poll.

Features include:

Swipe right/left on voting page when using the app on a phone
Intant camera upload to Amazon Web Services
Twilio allowing for texts to directly update database

Round Table

Ruby on Rails / JavaScript / Travis CI

Round table was a mock client project, built in 2 days by a team of 4 where we practiced client interaction from a tech team perspective. We were tasked to build out an application to the client's specifications, with input from our team. The app itself is very similar to wikipedia as it is a collection/list of board games that people can look up and play. A user can create a game by providing its title and instructions on how to play it. Adhering to proposed deadlines (MVP showcase) with the client changing requirements mid-way through were also simulated.

Features include:

Registered users can create the initial game and instructions
Images can be uploaded when creating a game via CKeditor/Paperclip gems
All users, registered or not can update a game's instructions & rules
All changes are recorded in a history and can be reverted
