Squad

Ruby on Rails / JavaScript / Twilio API / AWS S3 / Travis CI / React Native

Squad was our capstone project for Dev Bootcamp, built in 8 days by a team of 4. Our idea was to build a mobile friendly application that allows a user to create a friends list and send out polls/questions when they're out and about, needing quick advice. The user is then able to watch the feedback come back in real time via chart update.

Round Table

Ruby on Rails / JavaScript / Travis CI

Round table was a mock client project, built in 2 days by a team of 4 where we practiced client interaction from a tech team perspective. We were tasked to build out an application to the client's specifications, with input from our team. The app itself is very similar to a wikipedia with its features. Adhering to proposed deadlines (MVP showcase) with the client changing requirements were also simulated.
